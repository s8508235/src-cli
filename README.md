## Example
```sh
# From argument
text-to-video --text "Hello, World!"

# From pipe
echo "Hello!" | text-to-video -o output.mp4

# Custom styling
text-to-video -t "Big News" --font-size 72 --text-color yellow --bg-color blue -d 10
```
